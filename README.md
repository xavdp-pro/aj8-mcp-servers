# AJ8 MCP Servers

Custom MCP (Model Context Protocol) servers for AJ8 multi-agent chatbot.

## Servers

### 1. mcp-whatsapp (Baileys)

WhatsApp integration using Baileys (WhatsApp Web API).

**Features:**
- Send/receive text messages
- Send/receive images, videos, documents
- **Voice notes (PTT)** - auto-download `.ogg` files
- Webhooks for real-time messages
- Group chat support

### 2. mcp-sms-partner

SMS integration using SMS Partner (French SMS API).

**Features:**
- Send SMS (single and bulk)
- Check credits balance
- Message status tracking
- SMS history
- STOP list management
